## #Identificação do Projeto

**Tema:** Sistema de Gestão de Locadora de Veículos (Desktop)
**Linguagem:** Java (JDK 17+)
**Frameworks:** Hibernate (JPA), Java Swing, MySQL

## Membros e Responsabilidades

A equipe está organizada seguindo o padrão de arquitetura **MVC (Model-View-Controller)**, dividindo as tarefas conforme abaixo:

### 1. [Gustavo Semilio] (Líder do Projeto / Back-end)

- **Responsabilidade Principal:** Camada **Controller** e Regras de Negócio.
- **Tarefas:**
    - Implementação da lógica de cálculo de multas (Regras de 6h/12h).
    - Gerenciamento do repositório GitHub (Merge/Review).
    - Integração entre as telas (View) e o banco de dados (Model).
- **Stack:** Java Core, Lógica Algorítmica.

### 2. [Lucas Pessoa] (Front-end / UX)

- **Responsabilidade Principal:** Camada **View** (Interface Gráfica).
- **Tarefas:**
    - Desenvolvimento das telas no **Apache NetBeans (GUI Builder)**.
    - Criação dos Dashboards de status da frota.
    - Garantia da usabilidade para o atendente (Fluxo de telas).
- **Stack:** Java Swing, NetBeans Matisse.

### 3. [SEM AINDA] (Banco de Dados / Documentação)

- **Responsabilidade Principal:** Camada **Model** e Persistência.
- **Tarefas:**
    - Modelagem do Banco de Dados **MySQL**.
    - Configuração do **Hibernate (JPA)** e mapeamento das entidades.
    - Redação da documentação (Contrato, Orçamento, Manual).
    - Configuração dos relatórios em **JasperReports**.
- **Stack:** SQL, Hibernate, JasperSoft.
