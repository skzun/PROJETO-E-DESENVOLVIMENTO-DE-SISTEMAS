### CLÁUSULA 1ª - DO OBJETO

O presente contrato tem por objeto o desenvolvimento de um **Software de Gestão de Locadora de Veículos (Desktop)**, desenvolvido em linguagem Java e banco de dados MySQL, com a finalidade de gerenciar locações, clientes e manutenção de frota.

### CLÁUSULA 2ª - DAS OBRIGAÇÕES DA CONTRATADA

A CONTRATADA compromete-se a entregar:

1. Módulo de Cadastro de Clientes e Veículos (CRUD).
2. Módulo de Locação Imediata com verificação de disponibilidade.
3. Sistema de cálculo automático de valores (diárias + multas por atraso conforme regras de 6h/12h).
4. Funcionalidade de geração de contrato em PDF para impressão.
5. Código fonte e script de banco de dados para instalação local.

### CLÁUSULA 3ª - DAS LIMITAÇÕES DO ESCOPO (O que NÃO está incluso)

Fica expressamente acordado que o sistema **NÃO** contemplará:

1. Sistema de Reservas Futuras ou Agendamento (o software opera apenas com status atual/imediato).
2. Integração com gateways de pagamento real (cartão de crédito/débito online).
3. Versão Mobile ou Web (o software é exclusivo para uso Desktop/Computador Local).
4. Emissão de Nota Fiscal Eletrônica.

### CLÁUSULA 4ª - DAS OBRIGAÇÕES DA CONTRATANTE

A CONTRATANTE compromete-se a:

1. Fornecer os dados reais da frota para cadastro inicial.
2. Disponibilizar computador com sistema operacional Windows ou Linux para instalação.
3. Realizar os testes de validação em até 7 dias após a entrega da versão beta.
